# 人物设定：蓝清幽

## 基本信息
- **姓名**：蓝清幽
- **性别**：女
- **年龄**：23岁（穿越时）
- **身高**：168cm
- **出身**：蓝星地球，中国

## 外貌特征
- 中等身材，不胖不瘦
- 黑色长发，后期因魔女血统变成红色眼睛
- 常穿实用型服装，后期换上魔女套装
- 给人第一印象：普通但眼神坚定

## 性格特点

### 正面特质
1. **务实理性**：凡事以利益和效率为先
2. **善于规划**：喜欢做长远计划和风险评估
3. **学习能力强**：能快速掌握新知识
4. **护短**：对认定的"家人"极度保护
5. **执行力强**：决定了就立即行动

### 负面特质
1. **金钱至上**：过于看重利益
2. **有点中二**：喜欢给自己加戏和起奇怪的称号
3. **记仇**：对敌人从不手下留情
4. **腹黑**：喜欢设局坑害敌人

## 价值观

### 核心信念
- "活下来才是硬道理"
- "挡我赚钱者，死"
- "家人只有我能欺负"

### 处世原则
1. 不主动惹事，但不怕事
2. 利益分配要公平
3. 恩怨分明，有恩必报，有仇必还
4. 能用钱解决的问题都不是问题

## 成长轨迹

### 初期（1-50章）
- 被强制传送，慌乱但快速适应
- 发现炼金天赋，开始积累资源
- 认识到"养孩子"系统的价值
- 建立第一个稳定庇护所

### 中期（51-200章）
- 炼金术水平不断提升
- 收养更多神子，建立团队
- 开始接触更大的势力
- 参与区域联合和贸易

### 后期（201-500章）
- 成为魔女王级人物
- 建立魔女主城提丰
- 在种族大战中崭露头角
- 揭示世界真相的关键人物

### 终期（501-731章）
- 世界顶级强者
- 魔女联合体的领袖
- 对抗世界级威胁
- 寻找回家的方法

## 特殊能力

### 天赋进化
1. **轻身术（E级）**：身体轻盈，跳跃力增强
2. → **漂浮术（C级）**：可以短时间漂浮
3. → **漂移术（B级）**：可以短距离瞬移
4. → **星耀（SS级）**：高等级移动能力，具体能力待开发

### 魔眼
- **鉴析魔眼（S级）**：可以分析物品、生物的详细信息和弱点
- 后续进化：获得标记魔眼等其他魔眼能力

### 炼金术
- 从学徒级到宗师级
- 能炼制E-级到世界级药剂
- 开发多种原创配方

## 重要关系

### 神子/眷属
- **勺子**：寻宝鼠，宠溺有加
- **李怀**：忠诚部下，负责建设
- **商川**：商业代理，赚钱工具人
- **莉萨菈**：研究助手
- **艾丽莎**：魔女伙伴，研究伙伴

### 盟友
- **塔罗山庄**：古小贝等人
- **魔女族**：戴安娜、苏西、薇薇安等
- **死灵族**：结盟对象

### 敌人
- **奴隶主夏洛克**：前期反派
- **最高议会**：中期主要敌人
- **505编队**：敌对穿越者团队
- **驱魔人势力**：后期敌人

## 口头禅

1. "赚钱嘛，不寒碜"
2. "老娘就是做垄断生意"
3. "你这问题问得好，下一个"
4. "我是塔之魔女，我摊牌了"
5. "这特么给我干哪来了？"
6. "你搁着磕碜谁呢！"

1. 第45章：获得第一笔大额收入
2. 第60章：获得"高塔的魔女"称号

4. 第328章：成为"魔女王"
5. 第502章：建造飞行都市

**角色定位**：典型的"理智型+种田流"主角，通过智慧、规划和执行力，从零开始建立庞大势力。人物成长轨迹清晰，性格稳定但有变化，是整部小说的核心。
