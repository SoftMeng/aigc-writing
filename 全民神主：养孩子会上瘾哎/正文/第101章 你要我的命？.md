# 第101章 你要我的命？

合区后的第25天。

蓝清幽从塔罗山庄返回后，一直在思考渔者的预言。

【皇帝与月亮】

"皇帝意味着统治……"
"月亮意味着梦境……"

"这是什么意思？"

就在这时，她收到了一个意外的消息。

【系统消息】

【特殊事件：暗杀】

【目标：蓝清幽】

【刺客：？？？】

"暗杀？"

蓝清幽警惕。

【查看详情】

【刺客身份】

【姓名：影刃】
【等级：32级】
【职业：暗影刺客】
【称号：【影杀者】】

【任务：刺杀蓝清幽】

【奖励：1000万点数】

"1000万点数……"

"有人出这么大价钱买我的命。"

【准备迎战】

蓝清幽召集团队。

"准备战斗。"

【防御部署】

1. 【虫群大军×100】- 地下埋伏
2. 【四级魔力炮台×5】- 远程攻击
3. 【能量护盾】- 开启
4. 【护卫队×4】- 保护

【刺客到来】

【夜晚】

【时间：午夜】

【天气：月黑风高】

【袭击开始】

【影刃突袭】

【暗影步（S级）】

【效果：瞬移，无声无息】

【出现位置：蓝清幽卧室】

"你要我的命？"

蓝清幽冷静。

"拿去吧。"

【影刃攻击】

【暗影刺（S级）】

【伤害：5000点】

蓝清幽生命值下降。

【生命值：4500→0】

【死亡】

【复活】

【灵魂收容所：自动复活】

【消耗：1000灵魂】

【复活时间：1秒】

【复活完成】

【生命值：4500】

"你杀不死我。"

蓝清幽冷笑。

【反击开始】

【虫群突袭】

虫群从地下冲出。

【包围】

【酸液喷射+（C-级）】

【伤害：5000点/轮】

【影刃生命值：20000→15000】

"该死……"

影刃撤退。

【暗影遁（S级）】

【效果：逃跑】

"让他逃了。"

但蓝清幽已经得到了重要信息。

【刺客信息】

【雇主：？？？】

【线索：最高议会】

"最高议会？"

蓝清幽震惊。

"为什么他们要杀我？"

【本节完】
