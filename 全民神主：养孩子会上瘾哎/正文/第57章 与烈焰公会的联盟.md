# 第57章 与烈焰公会的联盟

蓝清幽通过魔法通讯联系了红莲。

【魔法通讯】
【对方：红莲（烈焰公会会长）】
【等级：19级】
【称号：烈焰之主】

"红莲会长，"蓝清幽说，"我接受您的结盟邀请。"

"太好了。"红莲高兴地说，"我立即带人过来。"

"现在就来吗？"

"对。"红莲说，"结盟仪式需要我们亲自到场。而且，我带来了礼物。"

"礼物？"

"你看了就知道了。"

三个小时后，烈焰公会的人马到达了幽月镇。

【烈焰公会代表团】
【人数：50人】
【等级：14-18级】
【领袖：红莲（19级）】

红莲是一个火红色长发的女性，气质优雅而强大。

【红莲】
【等级：19级】
【称号：烈焰之主】
【天赋：烈焰之体（A级）】
【武器：烈焰法杖（B+级）】

"蓝清幽小姐，"红莲微笑道，"很高兴见到你。"

"我也是。"蓝清幽说。

"我们开始结盟仪式吧。"红莲说。

【结盟仪式】
【流程：1. 宣读盟约 2. 交换信物 3. 共同宣誓】

"首先，宣读盟约。"

红莲拿出一份卷轴。

【盟约内容】
【双方：幽月镇（蓝清幽）& 烈焰公会（红莲）】
【条款：
1. 互不侵犯，互相支援
2. 共享资源和情报
3. 联合对抗共同敌人
4. 盟约有效期：1年】
【违约惩罚：赔偿10万点数+声望-100】

"盟约很公平。"蓝清幽说。

"是的。"红莲说，"我们烈焰公会从不占盟友的便宜。"

"我同意。"

"那我们交换信物。"

红莲拿出一个火焰形状的徽章。

【烈焰公会徽章（B级）】
【效果：在烈焰公会领地内，所有属性+10%】
【备注：盟友信物】

蓝清幽接过徽章。

然后，她拿出了自己的信物。

【幽月镇徽章（C级）】
【效果：在幽月镇内，所有属性+5%】
【备注：城镇信物】

"C级信物……"红莲笑道，"没关系，等级不重要，重要的是诚意。"

"最后，共同宣誓。"

两人同时宣誓：

"我，蓝清幽/红莲，在此宣誓，与烈焰公会/幽月镇结为盟友。互不侵犯，互相支援，共享资源和情报，联合对抗共同敌人。如有违背，甘愿受罚。"

【系统提示】
【盟约已生效】
【有效期：1年】
【双方成为盟友】

"结盟成功。"红莲高兴地说。

"谢谢。"蓝清幽说。

"现在，是礼物时间。"红莲说。

她挥手，身后的成员搬来几个箱子。

【礼物清单】
1. 【资源箱×10】（每个包含随机资源×1000）
2. 【装备箱×5】（每个包含随机C级装备×1）
3. 【技能书箱×3】（每个包含随机技能书×1）
4. 【特殊礼物：烈焰之心的邀请函】

"烈焰之心的邀请函？"蓝清幽疑惑。

"这是我们烈焰公会的核心建筑。"红莲解释，"烈焰之心可以提供特殊的训练，快速提升实力。我邀请您和您的核心成员来参加训练。"

"太感谢了！"

蓝清幽打开其他箱子。

【资源箱×10】
【获得：
- 精金×50
- 秘银×100
- 魔晶×200
- 各类药剂×500
- 抚养点数×50000】

【装备箱×5】
【获得：
- 烈焰铠甲（C级）×1
- 烈焰护手（C级）×1
- 烈焰长靴（C级）×1
- 烈焰项链（C+级）×1
- 烈焰戒指（C+级）×1】

【技能书箱×3】
【获得：
- 技能书-火焰护盾（C级）×1
- 技能书-烈焰冲击（B级）×1
- 技能书-火焰召唤（C级）×1】

"太丰富了……"蓝清幽震撼。

这些礼物的价值，至少超过20万点数！

"红莲会长太客气了。"

"不客气。"红莲笑道，"盟友之间，应该互相帮助。"

"我一定会报答的。"

"不用急。"红莲说，"来日方长。对了，还有一件事。"

"什么事？"

"关于世界神主大会。"红莲说，"我收到消息，大会上会讨论一个新的议题。"

"什么议题？"

"关于世界级道具的使用限制。"红莲说，"有些神主提议，世界级道具应该受到限制，不能随意使用。"

"限制？"蓝清幽皱眉。

"对。"红莲说，"如果这个提议通过，即使你合成了世界级道具，也会受到很多限制。"

"那怎么办？"

"所以，"红莲说，"你需要在大会上发言，反对这个提议。"

"但我才15级，没有投票权。"

"是的，但你可以列席发言。"红莲说，"如果发言有理有据，可能会影响投票结果。"

"我明白了。"蓝清幽说，"我会准备发言的。"

"很好。"红莲说，"另外，我们烈焰公会在大会上有投票权。我们会支持你的。"

"谢谢！"

"不客气。"红莲说，"好了，我们该回去了。烈焰公会还有很多事情需要处理。"

"再见。"

烈焰公会代表团离开。

蓝清幽清点礼物。

这些资源，足够她的团队再提升一个档次。

"立即分配装备。"

她将烈焰套装装备在自己身上。

【装备更新】
- 【精钢巨斧（C级）】
- 【烈焰铠甲（C级）】防御+50
- 【烈焰护手（C级）】攻击+20
- 【烈焰长靴（C级）】移动速度+10%
- 【烈焰项链（C+级）】智力+15，魔力恢复+5/秒
- 【烈焰戒指（C+级）】火焰抗性+30%

"全套装备，实力提升很多。"

她学习新的技能书。

【烈焰冲击（B级）】
【效果：向前方发射一道烈焰，造成150点火焰伤害，并点燃敌人（每秒20点伤害，持续5秒）】
【需求：智力30以上】

蓝清幽的智力是33点，满足条件。

【技能学习完成】

她又获得了一个强力技能！

"现在，去烈焰之心训练。"

她带上核心成员，前往烈焰公会的总部。

烈焰之心位于中央城南部的火山地区。

【烈焰之心】
【等级：A级建筑】
【功能：提供特殊训练，快速提升实力】
【训练项目：
1. 烈焰体质训练（体质+5，火焰抗性+20%）
2. 烈焰战斗训练（战斗技巧+20%）
3. 烈焰魔力训练（魔力上限+50）】

"太强大了……"

蓝清幽立即开始训练。

【训练开始】
【项目：全部】
【时间：7天】
【费用：免费（盟友特权）】

7天后，她的实力会再次提升。

"这次，我一定要变强。"

她暗暗发誓。
