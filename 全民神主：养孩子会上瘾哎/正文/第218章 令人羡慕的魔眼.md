# 第218章 令人羡慕的魔眼

【特殊发现】

战斗中发现特殊物品。

【掉落物品】

【物品：魔眼碎片（A级）】

【描述：可以制作魔眼】

【魔眼是什么】

【系统说明】

【魔眼：特殊器官】

【功能】

1. 【特殊视觉】- 可以看到特殊东西
2. 【魔法视觉】- 可以看到魔法
3. 【弱点洞察】- 可以看到敌人弱点
4. 【远视】- 可以看很远

【莉萨菈的反应】

"魔眼！"

莉萨菈惊讶。

"我一直想要魔眼。"

【莉萨菈的魔眼】

【当前技能：鉴定之眼（D+级）】

【效果：可以鉴定物品】

【想要升级】

"可以用魔眼碎片升级我的鉴定之眼。"

【蓝清幽同意】

"给你升级。"

【升级】

【使用：魔眼碎片（A级）】

【升级：鉴定之眼（D+级）→（B级）】

【新能力】

【新鉴定之眼（B级）】

【效果】

1. 【鉴定物品】- 可以鉴定物品
2. 【鉴定弱点】- 可以看到敌人弱点
3. 【鉴定等级】- 可以看到敌人等级
4. 【鉴定天赋】- 可以看到敌人天赋

【莉萨菈很高兴】

"太强了！"

【本节完】
