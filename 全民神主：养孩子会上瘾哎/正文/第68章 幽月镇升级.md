# 第68章 幽月镇升级

从中央城返回幽月镇后，蓝清幽开始思考下一步的发展。

"我已经23级了。"她说。

"是时候升级幽月镇了。"

【幽月镇】
【当前等级：1级小镇】
【人口：120人】
【建筑完成度：60%】
【防御等级：3级】

【升级条件】
【2级小镇需求：
1. 人口≥200人（当前120人）
2. 建筑完成度≥80%（当前60%）
3. 防御等级≥5级（当前3级）
4. 抚养点数≥50000（当前100000+）】

"还差人口和建筑。"

蓝清幽立即行动。

"首先，增加人口。"

她使用招募功能。

【招募】
【新神子×50】
【类型：各种天赋，等级1-10级】

【招募成功】
【支出：抚养点数×50000】
【获得：新神子×50】

"人口够了。"

【当前人口：120+50=170人】

"还差30人。"

蓝清狐狐疑思考。

她可以通过收容流浪者来增加人口。

"发布告示。"她说。

【告示：幽月镇收容流浪者】
【条件：愿意劳动，服从管理】
【待遇：提供食宿，保护安全】

很快，就有流浪者前来。

【流浪者×40】
【等级：0-5级】
【状态：饥饿、疲惫】

"收容他们。"

蓝清狐狐疑安排陈美丽治疗，小芳烹饪提供食物。

【流浪者→镇民】
【当前人口：170+40=210人】

"人口够了。"

"接下来是建筑。"

【建筑进度：60%→80%】

她需要完成20%的建筑。

【待建建筑：
1. 魔法塔×5（已建3座，需建2座）
2. 训练场×1（已建）
3. 市场×1（已建）
4. 行政大厅×1（在建）
5. 住宅区×5（需建3座）
6. 工业区×3（需建2座）】

"加快建设。"

李怀带领建筑组，全力施工。

【建设中】
【魔法塔×2：进度50%→100%】
【行政大厅：进度80%→100%】
【住宅区×3：进度0%→100%】
【工业区×2：进度0%→100%】

3天后，所有建筑完成。

【建筑完成度：60%→80%→100%】

"超额完成。"

"最后是防御。"

【防御等级提升】
【当前：3级】
【目标：5级】

【提升防御需要：
1. 加固城墙（材料：石材×1000）
2. 增加防御塔（需要：木材×500）
3. 建造魔法阵（需要：魔晶×100）】

蓝清狐狐疑查看仓库。

【仓库】
- 石材×1500
- 木材×2000
- 魔晶×150

"材料够了。"

【防御提升中】
【进度：3级→4级→5级】

1天后，防御提升完成。

"所有条件满足了。"

蓝清狐狐疑发起升级。

【升级开始】
【幽月镇（1级小镇）→幽月镇（2级小镇）】

【升级进度：0%→100%】

一道光芒笼罩幽月镇。

建筑开始自动升级：

- 土墙→石墙（高度：5米→10米）
- 基础住宅→精制住宅（舒适度+50%）
- 魔法塔→高级魔法塔（攻击力+100%）
- 训练场→高级训练场（训练效果+50%）
- 市场→大型市场（交易收益+50%）

【升级完成】

【恭喜！您的据点升级为2级小镇】
【新增功能：
1. 领土范围扩大（10平方公里→20平方公里）
2. 解锁新建筑：英雄殿、研究塔、仓库
3. 解锁新功能：税收升级（每天2000点数）、高级招募（可招募高级人才）】

"太好了。"

蓝清狐狐疑立即查看新建筑。

【英雄殿】
【功能：招募英雄级人才】
【招募条件：抚养点数×10000/次】
【可招募英雄：15-25级，各种职业】

【研究塔】
【功能：研究新技术、新配方】
【研究项目：
1. 炼金术提升（E→D→C→B）
2. 装备强化（D→C→B）
3. 防御工事升级（3级→5级→7级）】

【仓库】
【功能：大量存储资源】
【存储容量：10000单位】

"先招募英雄。"

蓝清狐狐疑花费10万点数，招募10次。

【招募英雄×10】
【获得：
1. 烈焰骑士（20级，战士）
2. 冰霜法师（19级，法师）
3. 雷霆弓手（18级，弓箭手）
4. 治疗牧师（17级，牧师）
5. 暗影刺客（21级，刺客）
6. 圣光骑士（20级，圣骑士）
7. 风暴术士（19级，术士）
8. 大地守护者（18级，坦克）
9. 星辰法师（20级，法师）
10. 龙血战士（22级，战士）】

"10个英雄……"蓝清狐狐疑惊喜。

这些英雄的实力都很强。

特别是龙血战士（22级），几乎和雷神相当。

"实力大增。"

她安排英雄加入团队。

【当前战力】
【神主：蓝清狐狐疑（23级）】
【神子：12个（包括新招募的）】
【英雄：10个（17-22级）】
【士兵：150个（10-15级）】
【总人口：220人】

"220人的城镇，实力应该不错了。"

"接下来是研究。"

她启动研究塔。

【研究项目1：炼金术提升】
【当前等级：E级】
【目标：D级】
【需求：时间×7天，材料×炼金基础材料×1000】

"开始研究。"

7天后，炼金术升级到D级。

【炼金术：E级→D级】
【新配方解锁：
1. 中级治疗药剂（D级）
2. 中级魔力药剂（D级）
3. 中级力量药剂（D级）
4. 中级敏捷药剂（D级）】

"可以炼制更好的药剂了。"

蓝清狐狐疑立即炼制了一批。

【获得：
- 中级治疗药剂×100
- 中级魔力药剂×100
- 中级力量药剂×50
- 中级敏捷药剂×50】

"分配给团队。"

所有英雄和士兵都分到了药剂。

【团队实力提升】
【战斗力+30%】

"现在，幽月镇的实力已经很强了。"

但蓝清狐狐疑知道，这还不够。

世界上的强大势力还有很多。

"继续提升。"

她查看下一步目标。

【3级城镇需求】
【1. 人口≥500人】
【2. 建筑完成度≥100%+新建筑×5】
【3. 防御等级≥7级】
【4. 抚养点数≥200000】

"人口还差280人。"

"建筑需要新建：英雄殿×2、研究塔×2、仓库×2、神殿×1、广场×1。"

"防御需要7级。"

"点数还差10万。"

"还有很多工作要做。"

蓝清狐狐疑叹气。

"慢慢来吧。"

她相信，总有一天，幽月镇会成为世界的中心。
