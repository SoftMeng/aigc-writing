# 第59章 各方震惊

魔力潮汐第8天，世界公告发布后，整个世界沸腾了。

【世界公告】

【神主蓝清幽建立了虫群大军】
【规模：100只C-级钻地蠕虫】
【位置：东部森林】
【威胁等级：高】

【各方势力收到消息】

【华夏区】

【联邦政府】
"100只C-级魔物……"
"这个神主不简单。"
"需要关注。"

【各大公会】
"虫群大军？"
"这是什么能力？"
"可以控制魔物？"
"太可怕了。"

【顶级神主们】
"东部森林的蓝清幽……"
"我听说过她。"
"才17级就有这种实力。"
"未来可期。"

【欧美区】

【光明教会】
"魔物控制……"
"这是黑暗魔法。"
"必须消灭。"

【暗黑议会】
"有趣的能力。"
"我们可以合作。"
"或者……消灭她，夺走虫群。"

【其他地区】

【各方反应】

1. 嫉妒
2. 恐惧
3. 贪婪
4. 警惕

【树洞】

蓝清幽收到了大量消息。

【系统消息】
【您有999+条新消息】

"这么多？"

她打开消息列表。

【消息分类】

1. 友好问候：100条
2. 威胁警告：200条
3. 合作邀请：300条
4. 收购请求：150条
5. 敌对宣言：50条
6. 其他：200条

"太多了……"

蓝清幽只看重要的。

【合作邀请】

【来自：铁血盟（华夏区前10公会）】
"蓝神主，我们希望合作。"
"我们可以提供资源，您提供虫群协助。"
"收益五五分成。"

"铁血盟……"

蓝清幽搜索了一下。

【铁血盟】
【排名：华夏区第8】
【会长：铁血战神（A级）】
【成员：10000+】
【实力：强】

"铁血盟实力不错。"

"但合作可能会失去控制权。"

她拒绝。

【拒绝理由】
"虫群只听命于我，无法外借。"

【下一个】

【来自：暗影公会（华夏区前20公会）】
"蓝神主，我们愿意高价收购虫群控制方法。"
"价格：100万点数。"

"100万点数……"

这个价格很诱人。

但蓝清幽思考。

"虫群控制是我的核心机密。"
"不能出售。"

拒绝。

【下一个】

【来自：神秘人】
"蓝神主，我知道你的秘密。"
"世界之水、月亮精灵……"
"我可以保密，也可以公开。"
"选择权在你。"

"威胁……"

蓝清幽皱眉。

"这个人知道我去了月亮湖。"

"他是谁？"

她查看对方信息。

【用户：？？？】
【等级：？？？】
【位置：？？？】

"隐藏信息……"

"可能是高级神主。"

蓝清幽回复。

"你想怎么样？"

对方回复。

"很简单。"
"我要世界之水的来源。"
"告诉我月亮湖的位置，我就保密。"

"交易？"

蓝清幽思考。

"月亮湖已经暴露了。"
"精灵实力提升到S级。"
"告诉他也无妨。"

"但需要交换。"

她回复。

"我可以告诉你。"
"但你需要提供等价信息。"

"什么信息？"

"关于魔力潮汐的后续事件。"

对方沉默了一会儿。

"好。"
"成交。"

【交易完成】

蓝清幽提供：月亮湖位置信息
对方提供：魔力潮汐后续事件

【魔力潮汐后续事件】

【第9天：兽潮大爆发】
【规模：5000+魔物】
【等级：C级为主】
【特殊：兽王出现】

【第10天：魔物进化】
【部分魔物进化为C+级】
【新技能出现】

【第11天：最终决战】
【兽王集结】
【世界BOSS可能苏醒】

"第9天有兽潮大爆发……"

蓝清幽思考。

"5000+魔物，C级为主。"

"而且有兽王。"

"必须做好准备。"

她召集团队。

"紧急会议。"

【会议开始】

"根据情报，第9天会有大规模兽潮。"
"5000+魔物，C级为主。"
"还有兽王。"

"我们需要加强防御。"

李怀发言。

"我可以建造更强的防御设施。"
"但需要材料。"

"什么材料？"

【高级防御设施】

【四级魔力炮台×5】
- 伤害：2000点/次
- 射程：500米
- 材料：秘银×100、魔石×500、能量核心×5

【能量护盾发生器×1】
- 防护：可以抵挡C+级攻击
- 持续时间：1小时
- 材料：能量核心×10、秘银×200

"能量核心……"

蓝清幽搜索交易市场。

【能量核心（C级材料）】
【价格：50000点数/个】

"太贵了……"

"需要15个，总共75万点数。"

"我们没有这么多点数。"

【当前点数：12万】

"还差63万。"

"有什么办法？"

商川发言。

"我可以筹集点数。"
"但需要时间。"

"多久？"

"3天。"
"我可以向其他商人借贷。"

"第9天就是兽潮大爆发。"
"来不及。"

蓝清幽思考。

"我们可以用其他方式。"

"比如，完成委托。"

她打开委托板。

【可用委托】

1. 【讨伐魔物】奖励：50000点数
2. 【采集材料】奖励：30000点数
3. 【护送商队】奖励：80000点数
4. 【探索遗迹】奖励：150000点数

"探索遗迹奖励最高。"

"但也很危险。"

"我们可以尝试。"

【探索遗迹】

蓝清幽决定。

"明天，我们去探索遗迹。"
"目标是获得能量核心。"
"同时完成其他委托。"

"虫群大军随行。"

"100只C-级蠕虫，足以应对大部分危险。"

团队同意。

【魔力潮汐第8天结束】

【收获】
- 觉醒神子×6
- 获得魔力潮汐情报
- 制定探索计划

【各方反应总结】

1. 联邦政府：关注
2. 铁血盟：合作邀请（已拒绝）
3. 暗影公会：收购请求（已拒绝）
4. 神秘人：威胁情报交易（已完成）
5. 其他势力：观望

"明天，探索遗迹。"
"为兽潮大爆发做准备。"

【本节完】
