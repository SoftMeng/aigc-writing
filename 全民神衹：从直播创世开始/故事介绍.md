# 《全民神衹：从直播创世开始》故事简介
李盛世，一个意外获得双穿能力的普通人，在蓝星与位面大陆之间架起了一座看不见的桥梁。("蓝星"和"位面大陆"双线发展。)
在位面大陆，十八岁觉醒成神是每个人的命运。神祇以眷族为刃，以信仰为粮，在无尽的位面碎片中征伐吞噬，只为让自己的神域更加辽阔。而李盛世发现，自己可以在两个世界自由穿梭——这是一个前所未有的优势。
他做了一个疯狂的决定：把神域伪装成一款跨时代的虚拟现实游戏。
从一场"直播创世"开始，《神域》横空出世。蓝星的游戏玩家们蜂拥而至，他们以为自己只是在玩一款设定精妙的兵种养成游戏——招募眷属、训练兵种、扩张地盘、开采资源。他们为了稀有兵种熬夜爆肝，为了领地扩张运筹帷幄，为了团队管理绞尽脑汁。
玩家们不知道的是：
他们指挥的每一场战斗，都在为李盛世攻伐真实的位面碎片；
他们开采的每一单位资源，都在壮大李盛世真实的神域；
他们培养的每一个"游戏角色"，都是李盛世真实的眷族；
他们以为的"版本更新"，其实是神域吞噬位面后的蜕变。
这是一场跨越两个世界的共谋。玩家们以创意为燃料，以热情为引擎，在不知不觉中成为李盛世最锋利的剑。而李盛世则隐于幕后，坐看第四天灾席卷位面大陆。
但随着游戏深入，变化悄然发生。当玩家在位面中成长，他们的提升开始渗透到蓝星——体魄的增强、能量的觉醒、技能的具现、智力的跃迁。游戏，正在改变现实。
资本嗅到了风口，国家看到了战略，科研人员在虚拟与真实的边界疯狂试探。蓝星因《神域》而繁荣，科技因智力跃迁而爆发。而李盛世的神域，也在玩家们的奇思妙想中，生长出连神祇都未曾想象过的兵种与可能。
从一个人到千万玩家，从一个神域到吞噬万界，从一款游戏到两个世界的变革——
这既是李盛世的成神之路，也是一群玩家的创世传奇。
当游戏成为真实，当玩家成为神话，谁才是被玩的那个？
烟火人间，星光神域。不求波澜万丈，但求一隅安宁——可这一隅，终成了两个世界的风暴眼。

## 必读设定
1. 神域可以进行时间加速，初期最高5倍时间加速。
2. 神域是独立的位面，通过位面信标，可以实现“神域A”和“神域B”的临时连接，然后进行“战斗”或者“采集”等。
3. 神域可以通过位面信标，连接搜索到的无主神域位面，然后进行“征讨”、“探索”、“吞噬”。无主神域位面可能有原生物种、宝藏、资源、其他。无主神域位面的会有一些评级，但是不太准。
4. 眷属的能力多样。
5. 蓝星在初期提供了廉价商品。在玩家加入神域后，会提供智慧的加持（玩家脑子灵活）。在蓝星人族得到神域能力反馈后，迎来"蓝星"和"神域"的大发展。
